# 📘 Guide Git : Collaboration à 3 avec Fusion dans la branche `main`

## 🛠️ Prérequis
- Avoir `git` installé
- Avoir accès au dépôt distant (ex. : GitHub)

```bash
git clone <URL_DU_DEPOT>
cd <nom_du_dossier>
```
